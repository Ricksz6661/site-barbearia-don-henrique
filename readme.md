# Don Henrique Barbearia

## Descrição
Este projeto é um site para clientes e um painel de administração de uma barbearia, desenvolvido em **HTML, CSS e JavaScript**. O site permite que os clientes agendem horários, escolham serviços e façam pagamentos via Pix, enquanto o painel do administrador gerencia serviços, clientes, horários, agendamentos, lucros e feedbacks.

## Funcionalidades

### Site do Cliente
- Agendamento online de serviços.
- Seleção de serviços disponíveis e cálculo automático do valor total.
- Visualização de horários disponíveis para agendamento.
- Pagamento via Pix com QR Code e chave Pix copiável.
- Envio de feedback com nome, comentário e avaliação por estrelas.

### Painel do Administrador
- Gerenciamento de serviços: adicionar, editar e excluir.
- Gerenciamento de clientes: adicionar, editar e excluir.
- Gerenciamento de horários disponíveis.
- Visualização de agendamentos e cálculo de lucros diário, semanal e mensal.
- Download de relatórios em PDF.
- Visualização e resposta de feedbacks enviados pelos clientes.
